# SQL로 하는 데이터 분석
* SQL 코딩테스트 : 서류 전형 이후 지원자 스크리닝 목적으로 진행
  * 해커랭크, 리트코드, 프로그래머스 medium 이상의 난이도
* SQL 쿼리 작성 순서
  * SELECT 조회 열
   * DISTINCT * : 중복된 값 제거
  * FROM 조회 테이블
  * WHERE 조건 (EX. WHERE age > 30)
  * GROUP BY 그룹화
  * HAVING 그룹에 대한 조건
  * ORDER BY 정렬
  * LIMIT 행 수 제한
